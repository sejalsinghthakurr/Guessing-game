# Number Guessing Game 🎲

An interactive logic-based game built with **Vanilla JavaScript**. This project focuses on mastering control flow, loops, and user input handling.

## 📌 Project Overview
This repository contains a functional "Guess the Number" game. The logic is designed to challenge the user to find a randomly generated secret number while providing real-time feedback.

> **Note:** Files are numbered (e.g., `11_guessingGame.js`) to align with my structured learning path in the **Sigma 4.0 Full-Stack Web Development** curriculum.

## 🛠️ Technical Concepts Applied
* **Randomization:** Utilizing `Math.random()` and `Math.floor()` to generate dynamic game states.
* **Control Flow:** Implementing `while` loops to maintain game state until the "win" condition is met.
* **Conditional Logic:** Using `if-else` blocks to provide "Higher" or "Lower" hints to the user.
* **User Interaction:** Handling browser-based prompts and console logging for a seamless experience.

## 🚀 How to Run
1. Open `00_index.html` in any modern web browser.
2. Open the **Developer Console** (`F12` or `Ctrl+Shift+I`).
3. Follow the prompts to enter your guesses and try to find the number in the fewest attempts possible!
